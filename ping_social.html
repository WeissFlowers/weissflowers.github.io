Technologies Used

NodeJS
- Server Side Language
ExpressJS
- Website back-end: Routing, Api handling
Handlebars
- HTML templating language which allows for easy population of common elements
MongoDB
- NoSQL database
-- User-Model: Profile picture, display name, username, privacy setting
-- Team-Model: display name, username, foreign-key for members, privacy setting
-- Ping-Model: Timestamp, foreign-key for User or Team, foreign-key for replies, likes
-- Notification-Model: foreign-key for recipient, foreign-key for Ping, foreign-key for user, action taken
SocketIO
- Facilitates realtime communication
-- Sends notifications to users in realtime if they are online
-- Displays online status of users
-- Keeps track of uptime of users
Google oAuth
- Allows users to log in using existing google account
- Users could change their usernames and display names after creating account

Features

Sending "Pings"
- Our version of tweets, gives the user a way of putting their thoughts and ideas onto the website
- Can reply to pings to create a thread
Private Account
- Allows the user to specify if their account is private and makes their pings only visible to followers
Realtime Noticiations
- SocketIO allows for the server to send notifications to users without the need to refresh the page
Online status Tracking
- SocketIO keeps track of user uptime so that it can be used for metrics or a 'leaderboard' type system
Team Accounts
- Works similar to a user account, but all team members can send pings from the account
