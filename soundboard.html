Technologies Used

NodeJS
- Central programming language and framework for creating the desktop webapp
Electron
- Allows for webpage to be used as a desktop webapp
- Looking for a more lightweight option
Voicemeeter
- 3rd party software that allows for creation of a virtual microphone
- Any virtual microphone software should work for this program

Features

Play Audio Through Virtual Microphone
- Allows the user to specify which output device to send the audio through
- Works with Virtual Microphone software like Voicemeeter to send audio through the virtual microphone
Record Desktop Audio
- Using Electron's apis, the user can record their desktop audio to create soundboard clips
- Circular queue allows the user to save the last 1-5 seconds of audio
-- Audio is then reconstructed in the nodejs program and saved
- User is also able to live record their desktop audio and save it the same way
- Clicking the mute button on the element allows for the circular recording to be paused
Add Audio To Soundboard
- Pressing the + button on the top left allows the creation of new soundboard elements
- These elements can then have audio files attached to them and given an option element image
-- The user can then play the soundboard element by either pressing the play button, or change the element to its image version and tap the image
Extendable Soundboard Elements
- Allows for a way of extending the current code to implement new types of soundboard elements
-- Some extended elements that I've created this way: Theme Music Element, Quick Clips Element, TTS Element
Theme Music Element
- Allows the user to specify theme music that plays in the background whenever they're microphone detects audio
- The user can change the audio, volume, and whether the audio loops
- The user can also decide if they wish to listen to the background music as well or only have it play through their microphone
- Pressing the mute button pauses the feature until it is pressed again
Quick Clips Element
- Element I created to allow for interfacing with arduino's using the serial port
- Allows for a physical array of buttons that can be pressed to make the soundboard more easily accessible
TTS
- Using an admittedly primitive text to speech engine to allow for funny TTS moments
- The user can only currently change the text to be spoken, not the model or modify any settings
